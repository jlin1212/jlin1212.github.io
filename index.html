Julia Set Renderer


Best for beefy browsers. Takes a Julia set and renders a 3D landscape. I may write some more about how this was done, but in brief the rendering is accomplished by raymarching a signed distance function derived from the Douady-Hubbard potential, which was used to define an implicit surface that could be refined with classic Mandelbrot-style iteration. Actually even writing that was a lot of work, so maybe not. Just look at the code if you really care.

Optimizer Sandbox - G R A D P I T T
