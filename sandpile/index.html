sandbox state: Saturated Every site in the sandbox is unstable, having a height of 4. Things look calm now, but starting the simulation will topple every site at once...

What happens if we start dropping grains of sand at random places on this state?

Notice that some drops trigger long cascades—avalanches—while others do almost nothing. In really rare cases, an avalanche can span the entire lattice, if a drop happens at the right spot. This spot is called a critical point.

You might think that eventually the pile settles down so that big avalanches become impossible at a certain point. But in fact, one is always possible: the system only gets more critical over time.
